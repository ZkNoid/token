# Zknoid Token

Implements initial version of $ZkNoid token

- Ticker: `$ZkNoid`
- Decimals: `9`
- Supply 1_000_000_000.0 tokens

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
